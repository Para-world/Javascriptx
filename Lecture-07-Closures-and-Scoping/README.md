## Mini Project: Console-based Contact Management System
A simple contact management system that runs in the browser console.

**Features:**
*   Add a new contact (name, phone, email).
*   View all contacts.
*   Search for a contact by name.
*   The project will be built using functions and objects, and will utilize closures to manage the contact list.
